# datawrangling_challenge
A Data Wrangling Challenge repository

## Submissions

This is a public repository and you don't have permissions to collaborate so, if you want to do it you have to:

1. Create a fork of this repo with your own github account.
2. Clone this repo in you localhost
3. Create a new branch named: submission/your_name
4. Create a new directory named: your_name inside "submissions"
5. Add your work to the new directory and commit your work
6. Push your updated branch to your fork
7. Create a PR (Pull Request)
8. Wait for approval
