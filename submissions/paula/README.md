En este proyecto se trabajó con un dataset que fue ensuciado manualmente de forma intencionada para simular problemas habituales en datos reales, como valores faltantes, filas duplicadas, valores extremos, errores de formato, tipos de datos incorrectos, errores tipográficos, categorías inválidas y nombres de columnas mal formateados. El objetivo fue identificar estos errores y corregirlos utilizando pandas, eliminando duplicados y filas vacías, imputando valores faltantes, limpiando símbolos y texto innecesario, corrigiendo formatos numéricos y tipos de datos, normalizando el texto y ajustando los nombres de las columnas. Tras el proceso de limpieza, el dataset quedó coherente, consistente y listo para su análisis.
